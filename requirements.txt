# =====================================================================
# LLM Wargame Middleware — pinned dependencies
#
# Air-gapped build rules:
#   - EXACT pins (==) only. No ranges. Reproducibility over freshness.
#   - Vendor these (and their transitive deps) as wheels on the download
#     network, then install offline in the Dockerfile:
#       pip download -r requirements.txt -d ./wheels \
#         --platform manylinux2014_x86_64 --python-version 3.12 \
#         --only-binary=:all:
#       (in Dockerfile) pip install --no-index --find-links=/wheels -r requirements.txt
#
# NOTE: pip download will also fetch TRANSITIVE dependencies (starlette,
# pydantic-core, anyio, sniffio, h11, httpcore, certifi, idna, typing_extensions,
# annotated-types, click, etc.) into ./wheels automatically. You do not list
# them here; they are pulled and vendored as part of resolving these top-level
# pins. Do NOT hand-pin transitives unless you hit a resolver conflict.
# =====================================================================
